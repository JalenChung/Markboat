# MarkBoat

---
这是一款简洁易用，自适应样式且可跨端的 Markdown 编辑器。可将`.md`文件格式转换为浏览器和微信可预览的`.html`和`.wxml`。

### MarkBoat 的特色⭐
* 多端适应
* 更加灵活的行号定位
* 分离式的目录和锚点
* 多端导出！多端共享！（此功能目前还在开发中）

### Welcome to pull develop together 😂（如果有的话）
* 本工程基于Uni-app(Vue3)编译运行建议使用HBuilder X
* 运行前
> npm i
* 目前主要开发分支为dev，后面考虑自己写解析库的话再去别的分支
